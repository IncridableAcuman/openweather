# Simple Weather Website with Java Spring Boot(Maven) and React(Vite)
* **
###### Note:
> ###### In this is project we will learn how to use RestTemplate
## 🚀 Quick Start
* **
### Prerequisites
* Java 21
* Node.js 18+,npm,pnpm
### Installation

```text
# Install dependencies
npm install
```
### Development

```text
# Start dev server
mvn spring-boot:run
```

```text
# Start dev server
npm run dev
```
App will be available at http://localhost:5173
