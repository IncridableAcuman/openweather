# 🌦️ OpenWeather

A modern weather forecast application that provides real-time weather information and forecasts for cities around the world using the OpenWeather API.

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Current temperature and weather conditions
* 💨 Wind speed information
* 💧 Humidity levels
* 🌅 Sunrise and sunset times
* 📅 Weather forecast
* 📱 Responsive design
* ⚡ Fast and user-friendly interface

## 🛠️ Tech Stack

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/IncridableAcuman/openweather.git
cd openweather
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file in the root directory:

```env
VITE_OPENWEATHER_API_KEY=your_api_key_here
```

### Run Development Server

```bash
npm run dev
```

Application will be available at:

```text
http://localhost:5173
```

## 🔑 Getting OpenWeather API Key

1. Create an account on OpenWeather.
2. Generate an API key.
3. Add the key to your `.env` file.
